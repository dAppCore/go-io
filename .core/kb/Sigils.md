# Sigils

Module: `forge.lthn.ai/core/go-io/sigil`

Cryptographic content addressing system. Sigils provide content-derived identifiers for files and data blobs, enabling deduplication and integrity verification.

## Files

- `sigil.go` — Core sigil type and operations
- `sigils.go` — Collection operations
- `crypto_sigil.go` — Cryptographic sigil generation (hashing)
